🍽️ Java Recipe Management System
A feature-rich, Object-Oriented console application built with Java to manage, search, and categorize culinary recipes.

🚀 About The Project
This project is a Command Line Interface (CLI) application that allows users to interactively manage a recipe database. It demonstrates the practical application of Object-Oriented Programming (OOP) principles and efficient data structures in Java.

✨ Key Features
Fast Searching: Implemented HashMap for O(1) time complexity in recipe search operations, ensuring instant results.
Smart Storage: Utilized Java Collections (LinkedList, ArrayList) to efficiently handle recipe storage and a dedicated "Favorites" feature.
Interactive CLI: Designed a user-friendly interface incorporating ANSI escape codes for terminal styling (e.g., color-coded difficulty levels).
Comprehensive Management: Users can seamlessly list all recipes, search by name, add new recipes, delete existing ones, and filter by categories (Breakfast, Main, Soup, Dessert).

🛠️ Technologies & Concepts Used
Language: Java
Core Concepts: Object-Oriented Programming (OOP), Data Structures
Collections Framework: HashMap, LinkedList, ArrayList
UI: Command Line Interface (CLI) with ANSI Colors
